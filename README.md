# AI-powered-Specialist-Recommendation-System
The AI-Powered -Specialist-Recommendation-System is a smart healthcare recommendation platform that analyzes patient data and automatically suggests the most suitable medical specialists. It aims to make diagnosis and referrals faster, accurate, and personalized by leveraging machine learning and medical records.

# Objective
Provide personalized specialist recommendations using AI
Minimize referral errors and delays
Improve patient care using data-driven decision making
Support personalized medicine using medical records, history, and genetics
Build a faster, accurate, and fully digital healthcare referral system

# Machine Learning Model Used
Decision Tree
Random Forest
Naive Bayes
